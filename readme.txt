Base de plugin propre pour TagAnimal :
- URL publique: /nfc/{slug} (modifiable dans Réglages)
- Si fiche inactive: formulaire d'activation
- Si active: fiche publique
- CPT ta_fiche, menu admin (Fiches, Outils, Réglages)

Installation:
1) Zipper le dossier taganimal-fiches et installer via WP > Extensions > Ajouter.
2) Activer le plugin, puis « Réglages > Permaliens > Enregistrer » pour régénérer les règles.

Extension future: exporter CSV, privacy, analytics, etc.
